=== SUPABASE STORAGE UI設定手順 ===

📋 手順1: バケット作成
1. Supabase Dashboard → Storage を開く
2. 「Create bucket」をクリック
3. バケット名: photos
4. 「Public bucket」をONに設定
5. 「Create bucket」をクリック

📋 手順2: RLS設定
1. photosバケットを選択
2. 「Policies」タブをクリック
3. 「RLS disabled」を選択
4. 保存ボタンをクリック

📋 手順3: 確認テスト
1. アプリで写真登録を試す
2. エラーが出なければ成功

=== 重要ポイント ===
- UIベースなので権限不要
- Public bucket = 誰でもアクセス可能
- RLS disabled = 開発用セキュリティ無効
- 本番ではRLSを有効化すること
